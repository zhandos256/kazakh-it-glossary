# kazakh-it-glossary

## Казақша IT терминдер аудармасы

program - қосымша (программаj)
variable - айнымалы
const variable - тұракты айнымалы
class - класс
function - функция
argument - аргумент
parameter - параметр
return - қайтару
key word - кілт сөз
database - деректер қоры
url - сітеме
programming language - бағдарламау тілі
result - нәтиже
array - массив
brackets - жақша
use - колдану
code - код
